# 日付処理

- date-fns

date-fnsは、Moment.jsよりも軽量なライブラリであり、TreeShakingに対応

## 日付のフォーマット変換

日付のフォーマット変換には、`date-fns` の `format` 関数を使います。

```ts 
import { format } from "date-fns"

console.log(format(new Date(), "yyyy/MM/dd")) // 2023/03/24
console.log(format(new Date("2023-04-02"), "yyyy年M月d日") // 2023年4月2日
```
